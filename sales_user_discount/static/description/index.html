Sales User Discount

User wise discount is providing the easiness in the sales system to finalized some level of discount.

The module gives the functionality to define user wise discount in particular customer. So, while creating the Sales Quotation or Sales Order in the order line it will apply the defined discount for the customer which was finalized as per the particular user.

To apply this functionality, primary need to allow the discount on the order line because it will allow the discounts in the Sales Quotation or Sale Order.

The below image shows, configuration access to apply the functionality of the sales user discount.

If access User: User Discount then the user can only see its own applicable discount in the read only mode and if access Manager: User Discount then the user can see all users applicable discount and also create and modify the discount value in particular customer for any user.

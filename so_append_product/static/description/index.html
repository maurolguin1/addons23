Enjoy Easy Ordering

Overview

This module will help you to order products easily. You can create quotations and you can add products without navigating to the quotations menu. Added new customized feature in the view of product variants. That will help you to perform order by looking the catalog.

Added new customized button 'Add to Quotation' in the kanban view of product variants.

Button opens a wizard. Either here you can choose an existing quotation or you can create new quotation. Then enter Quantity, Unit price, Taxes etc. and click insert button. Now your product will be inserted in the particular quotation. Then you can order next product.

This button is also available in the form view.
